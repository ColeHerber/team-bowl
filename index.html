Project Overview

Team Bowl is a six-legged wheeled robot platform built around a Jetson AGX Orin compute module. It combines the stability of legged locomotion with the efficiency of wheel-based driving, enabling versatile navigation across unstructured terrain.

System Architecture

The robot is organized into modular ROS 2 packages spanning locomotion, perception, safety, and autonomy. Each subsystem communicates over a shared message bus with well-defined interfaces.

System Architecture Diagram

Locomotion

Leg controllers and velocity multiplexer coordinate six RS04 joints with wheel drive motors, managed by VESC motor controllers.

Perception

An OAK-D stereo depth camera provides real-time RGB-D data. A depthai-ros integration streams point clouds and obstacle data into the autonomy stack.

Safety

A hardware e-stop and software heartbeat monitor ensure safe operation. A collision guard layer prevents locomotion commands that would cause contact.

State Estimation

An Xsens MTi IMU provides 6-DOF orientation and acceleration. Fused with leg kinematics for reliable odometry on rough terrain.

Planning

Autonomous planning stack takes high-level goals and generates waypoints, obstacle-avoiding paths, and velocity commands.

Simulation

Full MuJoCo simulation environment mirrors the physical robot for rapid controller development and validation before hardware deployment.
